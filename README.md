# Project3_CellularAutomata
Python code showing cellular automata

## How to run
1. Download .py file
2. On terminal, make sure matplotlib is downloaded (if not, run **pip install matplotlib**)
3. Run **python cellularAutomataProj3.py** on terminal

## Other notes
- You can edit N to change the size of the grid
- Under anim, you can change the interval to be faster or slower
